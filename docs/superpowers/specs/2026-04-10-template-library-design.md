# 模板库 + 节点面板改版 设计文档

**日期：** 2026-04-10  
**范围：** 新增模板库 Tab、ImageNode / VideoNode 控制面板 UI 改版、AI 融合提示词

---

## 1. 背景与目标

用户在画布上使用图片节点和视频节点生成内容时，每次都需要从零手写提示词，效率低。通过引入行业模板库，用户可选择预设的题材模板（古风武侠、都市情感等），AI 将模板提示词与用户补充描述自动融合，直接输出可用的生成提示词。

---

## 2. 数据结构

新增 SQLite 表 `templates`：

```sql
CREATE TABLE templates (
  id           TEXT PRIMARY KEY,
  name         TEXT NOT NULL,        -- "古风·开场航拍"
  genre        TEXT NOT NULL,        -- 古风武侠 / 都市情感 / 科幻奇幻 / 微短剧爆款 / 自定义
  nodeType     TEXT NOT NULL,        -- "image" | "video"
  promptPreset TEXT NOT NULL,        -- 核心提示词预设

  -- 图片模板专属（nodeType = "image"）
  styleTag        TEXT,              -- 画风建议，如"中国水墨/暖色调"
  compositionTip  TEXT,              -- 构图建议，如"三分法/低角度仰拍"

  -- 视频模板专属（nodeType = "video"）
  cameraParams    TEXT,              -- 运镜参数，如"缓慢推进+俯拍45°"
  durationHint    INTEGER,           -- 建议时长（秒），如 5
  audioHint       TEXT,              -- 音乐/音效建议

  createdAt    INTEGER NOT NULL
);
```

---

## 3. 后端 API

| 方法   | 路径                          | 说明                                          |
|--------|-------------------------------|-----------------------------------------------|
| GET    | `/api/templates`              | 列表，支持 `?nodeType=image&genre=古风武侠` 筛选 |
| POST   | `/api/templates`              | 新建模板                                       |
| PUT    | `/api/templates/:id`          | 编辑模板                                       |
| DELETE | `/api/templates/:id`          | 删除模板                                       |
| POST   | `/api/templates/merge-prompt` | AI 融合：模板提示词 + 用户输入 → 优化提示词      |

### merge-prompt 请求/响应

```ts
// POST /api/templates/merge-prompt
Request: {
  templatePrompt: string;   // 模板的 promptPreset
  userInput: string;        // 用户补充描述（可为空）
  nodeType: 'image' | 'video';
  shotDescription?: string; // 节点的分镜描述（如有）
}

Response: {
  mergedPrompt: string;     // AI 融合后的提示词
}
```

AI 系统提示词要求：将模板提示词作为风格基底，融入用户的个性化描述，输出一段适合图像/视频生成的专业英文提示词（中文界面展示中文版本）。

### 4.1 入口

底部 Tab 栏新增「模板库」Tab，图标使用 `BookMarked`（lucide），位于现有 Tab 之后。

## 5. ImageNode 面板改版

选中图片节点（单选）时，节点下方展示控制面板，从上到下：

```
┌─────────────────────────────────────────────┐
│ 模板 [全部▾] [✦ 古风·开场航拍] [都市·特写] +5 │  ← 横向可滑动
├─────────────────────────────────────────────┤
│ ✦ AI 融合提示词（可直接编辑）          [编辑]  │
│  古风山水，航拍俯视，晨雾缭绕，竹林深处…       │
├─────────────────────────────────────────────┤
│ + 加入你的想法（可选，textarea）              │
├─────────────────────────────────────────────┤
│ 参考图: [缩略图] [+]                         │
├─────────────────────────────────────────────┤
│ 画风: [写实✓] [动漫] [油画] [水墨] [自定义…] │
├─────────────────────────────────────────────┤
│ SeeDream 4.5 | 16:9·2K▾      [1×] [生成↑]  │
└─────────────────────────────────────────────┘
```

**交互流程：**
1. 用户点击模板 chip → 立即调用 `POST /api/templates/merge-prompt`（templatePrompt + 当前 userInput）
2. 返回结果填入「AI 融合提示词」区域（可编辑 textarea）
3. 用户也可在「加入你的想法」中补充，blur 时重新触发融合
4. 点击「生成↑」使用融合提示词调用图片生成 API

**兼容性：** 无模板选中时，「AI 融合提示词」区域退化为普通提示词输入框，保持原有行为。

## 6. VideoNode 面板改版

```
┌─────────────────────────────────────────────┐
│ [文生视频] [图生视频]                         │
├─────────────────────────────────────────────┤
│ 模板 [全部▾] [✦ 科幻·飞船穿越] [古风·追逐] +4│
├─────────────────────────────────────────────┤
│ ✦ AI 融合提示词（可编辑）              [编辑]  │
│  镜头缓慢推进，科幻星际战舰穿越大气层…          │
├─────────────────────────────────────────────┤
│ + 加入你的想法（可选）                        │
├─────────────────────────────────────────────┤
│ 参数: [16:9] [720p] [5s] [音频开] 推荐:缓推▾  │
├─────────────────────────────────────────────┤
│ Seedance 1.5 Pro              [1×] [生成↑]  │
└─────────────────────────────────────────────┘
```

**视频模板额外行为：** 选中视频模板后，若模板含 `cameraParams`，在参数行显示「推荐: {cameraParams}」标签（高亮紫色），点击可一键应用到提示词中；若含 `durationHint`，自动将时长选择器切换至推荐值。
