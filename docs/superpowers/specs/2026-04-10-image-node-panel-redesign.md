# 图片节点控制面板重设计

**日期**: 2026-04-10  
**文件**: `src/components/ImageNode.tsx`

---

## 概述

将图片节点的底部控制面板从「固定展开的大卡片」改为「紧凑底部工具栏 + 按需展开面板」的交互模式，参考图像生成工具的通行设计语言。

---

## 交互模型

面板分为三层：

```
┌──────────────────────────────────────┐
│  输入区（始终可见）                    │
│  - 上传按钮 + 提示词大文本区           │
│  - 镜头描述（有值时显示）              │
├──────────────────────────────────────┤
│  展开区（按图标切换，互斥，可收起）     │
│  - 🎨 风格模板面板                    │
│  - 👥 资产面板                        │
├──────────────────────────────────────┤
│  底部工具栏（始终固定）               │
│  模型 │ 比例 │ 清晰度 │ 🎨 │ 👥 │ ─ │ ✨ │ 1x │ ↑  │
└──────────────────────────────────────┘
```

点击 🎨 或 👥 图标：展开对应面板；再次点击或点另一个：收起/切换。两个面板互斥，不同时展开。

---

## 各区域详细设计

### 1. 输入区

- **上传按钮**：左上角虚线框（+ 上传），点击触发文件选择，支持拖拽放入
- **提示词**：右侧大文本区，placeholder "可复制或拖拽图片至此，描述你想要的画面"，支持 Enter 触发生成（Shift+Enter 换行）
- **镜头描述**：有 `data.shotDescription` 时在文本区下方显示，带左侧竖线，只读灰色文字，标注「镜头描述」前缀

### 2. 风格模板面板（🎨 展开）

#### 分类筛选行
- 来源：`/api/templates?nodeType=image` 返回数据中的 `styleTag` 字段去重
- 固定追加一个「+ 自定义」按钮，点击打开自定义模板输入
- 点击分类 pill 过滤下方网格；默认选中「全部」
- 原有的「写实/动漫/油画/水彩/赛博朋克/中国水墨/素描/3D渲染」全部迁移为模板库条目，不再作为硬编码 preset

#### 模板网格
- 5列网格，每格：缩略图（`previewImage` 字段，无图时显示 emoji 占位）+ 名称
- 点击选中（高亮紫色边框），再次点击取消
- 选中模板后：调用 `/api/templates/merge-prompt` 将模板提示词与用户输入合并，结果填入提示词区
- 最后一格固定为「+ 添加」，与模板库的自定义添加联动
- **空状态**：若 `/api/templates` 返回空数组，展示「暂无模板，点击 + 添加」提示，不显示分类筛选行
- **迁移说明**：原硬编码的「写实/动漫/油画/水彩/赛博朋克/中国水墨/素描/3D渲染」作为默认模板，由后端在数据库为空时自动预置（初始化 seed），前端不再硬编码

### 3. 资产面板（👥 展开）

- **分类 Tab**：角色 / 场景 / 道具（从 `AssetItem.category` 派生）
- **资产网格**：4列，显示 `assets` 中当前分类的资产缩略图 + 名称
- **点击资产**：将其 `src` 追加到参考图列表（`uploadedRefImages`）
- **「+ 创建」格**：跳转/打开资产管理页面

### 4. 底部工具栏

从左到右：

| 元素 | 类型 | 说明 |
|------|------|------|
| 模型选择 | 下拉 Pill | 当前 SeeDream 4.5，可扩展 |
| 比例 | 下拉 Pill | 1:1 / 4:3 / 3:4 / 16:9 / 9:16 / 3:2 / 2:3 / 21:9 |
| 清晰度 | 下拉 Pill | 1K / 2K，选中后同步 `quality` state |
| 🎨 风格图标 | 圆形按钮 | 激活时紫色背景，切换风格面板 |
| 👥 资产图标 | 圆形按钮 | 激活时紫色背景，切换资产面板 |
| `spacer` | 弹性间距 | — |
| ✨ AI 优化 | 小按钮 | 调用 `/api/optimize-prompt`，结果填入提示词区；无镜头描述时 disabled |
| 数量 | 下拉 Pill | 1x / 2x / 3x / 4x |
| 下载 | 图标按钮 | 有图片时出现，下载当前图片 |
| ↑ 生成 | 圆形按钮（紫/粉） | 无提示词时 disabled |

## 状态管理

新增/调整的 React state：

```ts
const [expandedPanel, setExpandedPanel] = useState<'style' | 'asset' | null>(null);
// 原 selectedStyle / customStyle → 统一为 selectedTplId（模板ID）
// 原 isRatioOpen / isCountOpen → 保留（下拉菜单开关）
// quality state 保留，新增 UI 控件
// uploadedRefImages 保留（资产点击追加到此）
```

`expandedPanel` 逻辑：
- 点击已激活的图标 → 设为 null（收起）
- 点击未激活的图标 → 设为对应值（互斥切换）

## 尺寸

- 面板宽度：`460px`（从原 520px 收窄，更贴合参考设计）
- 风格网格展开后面板增高约 220px
- 资产网格展开后面板增高约 160px

## 不变的部分

- 参考图（通过边连接传入 `data.referenceImage`）：保留逻辑，资产点击追加到 `uploadedRefImages`，上传按钮也追加
- 生成逻辑（`generateImages` API 调用）：不变
- 分镜打勾按钮：不变
- 多图切换胶囊（右下角）：不变
- 生成中流光边框动效：不变
- `showPanel = selected && selectedCount === 1` 逻辑：不变
